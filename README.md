# BashmentTech Landing Page

BashmentTech is an AI-powered analytics platform designed to help BIPOC event promoters predict attendance, optimize marketing, and increase revenue. This repository contains the dark-mode single-page landing site.

## Features
- Hero section with headline, subheadline, and CTA buttons
- Features grid: Attendance Forecasting, Marketing Analytics, Revenue Optimization, Community Network
- How it Works section: 3-step process
- Beta Promoters section highlighting 15 BIPOC promoters
- About section with mission, vision, and founder
- Footer with links and newsletter signup

## Usage
1. Upload the repository to GitHub.
2. Enable GitHub Pages under repository settings.
3. Visit your GitHub Pages URL or configure your custom domain (bashment.tech).
